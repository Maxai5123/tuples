# Проект: "Кортежи"

## Описание проекта
В этом пролекте наглядно показано как можно и нельзя работать с `tuples` (кортежами) - это коллекция данных которая не позволяет вносить какие-либо изменения в нее.

## Цель проекта
Показать отличия между кортежами и списками на живых примерах и короткой игре.

### Проект был создан в образовательных целях в рамках курса "Введение в Python" онлайн-школы "лидер".
